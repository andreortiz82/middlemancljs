## Getting started
A starter app using [Middleman](http://middlemanapp.com) (external pipeline protocol) and [Clojurescript](https://clojurescript.org/).

**Installing the deps**
`bundle install`
`yarn install` or `npm install`

**Running the app**
`middleman server`

Visit [localhost:4567](http://localhost:4567)
